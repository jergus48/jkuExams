## Question 1 — Spam Filter (Bayes' Theorem)

**Problem:** A spam filter works by having users training the filter. It looks for patterns in the words in mails marked as spam by the user. For example, it may have learned that the word "win" appears in 30% of the mails marked as spam. It can be assumed that 0.9% of non-spam mail includes the word "win" and 40% of all mails received by the user are spam.

- a) Calculate the total probability that the word "win" appears in a mail regardless the type of the mail. [15 points]
- b) Calculate the probability that a mail is not a spam mail, given the word "win" appears in it. [15 points]

**Solution:**

**a) Answer: 0.1254**

**b)**
- p(win | no spam) = 0.009
- p(win | spam) = 0.3
- P(spam) = 0.4
- P(no spam) = 0.6

P(no spam | win) = 0.009 * 0.6 / (0.009 * 0.6 + 0.3 * 0.4) = **0.043**

## Example 5a-b — Linear Regression (TV Time & Weight)

**Problem:** The television time in hours per day and the weight in kilograms of a population of 15 children were examined. The following results were obtained:

| | mean | SUM(x_i - x̄)² (i=1 to 15) | covariance |
|---|---|---|---|
| television time | 4.733 | 42.933 | 15.973 |
| weight | 52.400 | 2561.600 | |

Furthermore, it is known that there is a linear relationship between television time and weight.

- a) Using the above information, show that the variance of the television time is 2.862 h² and the variance of the weight 170.773 kg². (Note: numbers are rounded to 3 decimal places, the 15 children are considered as a population) [8 points]
- b) Determine a linear regression model to predict the weight of a child with 6 hours of television time. State the predicted weight and the parameters of the linear regression model. [12 points]

**a)**
- 1/15 * 42.933 = **2.8622**
- 1/15 * 2561.600 = **170.773**

- y = 25.985 + 5.581 * x_i
- 25.985 + 5.581 * 6 = **59.4715 kg**

## Question 2 — Conceptual Questions (Multiple Parts)

- 2a) Why plays the "normal distribution" such an important role in inductive statistics? [4 points]
- 2b) Explain with a few words, what is meant by a "type I error" and a "type II error" in inductive statistics. [4 points]
- 2c) Explain with a few words, what is understood under the term "p-value" in inductive statistics. [4 points]
- 2d) Explain with a few words, the difference between a metric-continuous and a metric-discrete attribute in descriptive statistics. State one example for each kind of these attributes. [4 points]
- 2e) Describe with a few words the advantage and disadvantage of the "arithmetic mean" and "mode" (= modal value) as measures of location. [4 points]

**Answers:**

**2a)** The central limit theorem tells us that a population with high sample size is normal distributed, many things in "Nature" are normally distributed (IQ, height of humans) and other distributions can be transformed into a normal distribution.

**2b)** A type I error is the rejection of a true null hypothesis ("false positive" finding); example: an innocent person is convicted. A type II error is the non-rejection of a false null hypothesis ("false negative" finding); example: a guilty person is not convicted.

**2c)** The p-value is the probability — under the null hypothesis — of obtaining a "result". (The p-value is the concrete type I error of a data situation.) If the p-value is high, there is a high likelihood of obtaining said result under the null hypothesis; therefore the null hypothesis is not rejected. If the p-value is small --> the null hypothesis is rejected.

**2d)** Metric discrete data = values from a fixed list of numbers (= counting apples on a tree). Metric continuous data = values can be determined with infinite precision (= measuring the weight of an apple in grams -> theoretically the weight can be measured with infinite precision).

**2e)** "Arithmetic mean" = disadvantages: can only be used for metric data, not stable against great outliers in the data. Advantages = if the data has no great outliers, it is a great measure of location (we get the middle of the data). "Mode" = disadvantage = must not be unique. Advantage = we get the attribute with the highest frequency.

## Exercise 2: Global and Local Extrema (15 points)

Consider the function f: R → R with f(x) = e^(-x²/2) · x.

**a)** Determine all local and global extrema and extreme values of f.

**b)** Where is f monotonically increasing/decreasing?

**c)** Where is f convex/concave?

**Solution:**

We have:
- f'(x) = e^(-x²/2)(1 - x²)
- f''(x) = e^(-x²/2) · x(x² - 3)

**a)** Setting f'(x) = 0: x₁ = 1 and x₂ = -1.
- f''(1) = -2/√e < 0 → **local maximum at x₁ = 1**, value f(1) = 1/√e
- f''(-1) = 2/√e > 0 → **local minimum at x₂ = -1**, value f(-1) = -1/√e

Since lim_{x→±∞} f(x) = 0:
- **Global minimum at x₂ = -1** with value f(-1) = -1/√e
- **Global maximum at x₁ = 1** with value f(1) = 1/√e

**b)** f is **decreasing** in (-∞, -1) ∪ (1, ∞) and **increasing** in (-1, 1).

**c)** f is **concave** in [-∞, -√3] ∪ [0, √3] and **convex** in [-√3, 0] ∪ [√3, ∞).

## Exercise 3: Taylor Series (7 points)

Determine the Taylor polynomial T₄(x) of order 4 of the function:

around x₀ = π.

**Solution:**

- f''(x) = 2(cos²(x) - sin²(x))
- f'''(x) = -8sin(x)cos(x)

- f''(π) = 2

## Exercise 5: Fourier Series (10 points)

Compute the Fourier series Sh(x) of the function h: [0, 1) → R with h(x) = 2x - 1.

Simplify the result! (Complex numbers in canonical form.)

Determine all x ∈ [0, 1) for which Sh(x) equals h(x).

**Solution:**

We compute Fourier coefficients according to Definition 7.14:

f̂(k) = 2∫₀¹ xe^(-2πikx)dx - ∫₀¹ e^(-2πik)dx = -1/(πik) = i/(πk) for k ≠ 0.

For k = 0: f̂(0) = 0.

Thus:
**Sh(x) = (i/π) · SUM_{k≠0} (1/k) · e^(2πikx)**

Since h: [0, 1) → R is piecewise continuous, by Dirichlet's theorem (Theorem 7.39):
- h is differentiable in x ∈ (0, 1), therefore for those x, **Sh(x) = h(x)**.
- For x = 0: h(0) = -1 but Sh(0) = 0 → **no convergence at x = 0**.

Therefore Sh(x) = h(x) for all x ∈ **(0, 1)**, but not at x = 0.

## Exercise 6: Extrema under Constraints (10 points)

Calculate the global extrema of:
f(x₁, x₂) = 2(x₂ - 1)² + (x₁ + 1)²

under the constraint:
x₁² + 2x₂² ≤ 1/3

**Solution:**

First: the only stationary point of f on R² is at (x₁, x₂) = (-1, 1), which does not fulfill the constraint. Hence no extrema in the interior Ω° = {x₁² + 2x₂² < 1/3}.

Use Lagrange multipliers on the boundary x₁² + 2x₂² = 1/3:

Lagrange function:
L(x₁, x₂, λ) = (x₁ + 1)² + 2(x₂ - 1)² + λ(x₁² + 2x₂² - 1/3)

∇_x L = (2(x₁ + 1) + λ2x₁, 4(x₂ - 1) + λ4x₂)

Solve:
- 2(x₁ + 1) + λ2x₁ = 0
- 4(x₂ - 1) + λ4x₂ = 0
- x₁² + 2x₂² = 1/3

From first two equations (λ ≠ -1):
- x₁ = -1/(1+λ)
- x₂ = 1/(1+λ)

Substituting into constraint:
(-1/(1+λ))² + 2(1/(1+λ))² = 1/3
→ 9 = (1+λ)² → 3 = |1+λ| → **λ = 2 or λ = -4**

Critical points:
- λ = 2: (-1/3, 1/3) → f(-1/3, 1/3) = **4/3 (global minimum)**
- λ = -4: (1/3, -1/3) → f(1/3, -1/3) = **16/3 ≈ 5.33 (global maximum)**
